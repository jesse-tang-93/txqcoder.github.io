MarkDown语法简单小记

发表于 2017-07-03

斜体和粗体

斜体用一个 * 包裹文字 例如 * 字* 字

粗体用两个 ** 包裹文字 例如 *字\* 粗体字

标题 h1-h6

标题一般用 # 的个数来显示是几级标题 h1=# 文字 h2=## 文字 以此类推。

在常用的MD编辑器例如Typora中可以使用ctrl+数字 的快捷键来书写标题 例如h1=Ctrl+1

文字加链接

链接用[文字](链接的url) 例如 点我去百度 [点我去百度](http://www.baidu.com)

列表

有序列表用 数字 + . +空格 的方式表示 例如 1. 我是有序列表

  1. 我是有序列表
  2. 注意数字和 . 之后要有一个空格

无序列表常用一个+ 来表示 例如+ 我是无限列表

  • 我是无序列表
  • 无序列表的+ 号之后也要一个空格

引用

引用一段话用> 例如> 表示引用

表示引用

行内代码块

行内代码用英文的反引号包裹 例如` code ` cde

代码片段用三个反引号包裹``` 例如

code
​

let a=”hi”
console.log(a)
typeof a
```

图片

使用![描述](图片url) 来实现图片的展示 例如

![百度logo](//www.baidu.com/img/bd_logo1.png)
