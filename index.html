入门vue第一天总结

vue安装步骤及常用命令

​ 1、npm install -g vue-cli //全局安装vue-cli

​ 2、 vue init webpack my-porjectName //生成自己的项目

​ cd my-porjectName //进入项目

​ 3、npm install 安装依赖

​ 4、npm run dev //启动项目

​ 5、npm run build //打包上线

打包完成后会生成dist文件，项目上线时只需要将dist文件夹放到服务器。

若要本地打开打包后的文件，需要在config> index.js下 将打包后的文件路径由原来的 ‘ / ’ 修改为‘ ./ ’ ，在这个页面也可以配置端口号

vue组件使用注意事项

  1. 每个组件下只能有一个最大包含的div，不能出现平级元素。也就是说所有的内容必须放在 template 下的div里。
  2. 每个组件在使用时必须先引入，再注册。
    import myTem from "./pages/myTem.vue" //引入
    components:{myTem} //在需要使用的页面的js export default{}中增加components属性，注册要使用的组件
  3. 组件在页面使用中用闭合的注册的组件名称标签 eg. <myTem></myTem>

vue常用指令

  • vue-text 文本显示指令
  • vue-html 将返回的html代码编译后展示在页面 eg. data(){return{html2:'<span>注意这里应该使用es6的模板语法反引号</span>'}}
  • v-on:click 事件绑定 简写为 @click=“ ”
  • v-bind 绑定属性 简写为 : src :class :date-id ….
  • v-model 双向数据绑定
  • v-once 文本只会赋值一次

vue的修饰符 一般放在事件之后。 .prevent 阻止默认事件的触发 .trim 去除用户输入收尾的空格 .native 为引入的组件响应添加的事件

什么JavaScript

JavaScript 是一种由 Netscape 的 LiveScript 发展而来的原型化继承的基于对象的动态类型的区分大小写的客户端脚本语言，主要目的是为了解决服务器端语言，比如 Perl，遗留的速度问题，为客户提供更流畅的浏览效果。

当时服务端需要对数据进行验证，由于网络速度相当缓慢，只有 28.8kbps，验证步骤浪费的时间太多。于是 Netscape 的浏览器 Navigator 加入了 JavaScript，提供了数据验证的基本功能。

JavaScript，一种直译式脚本语言，是一种动态类型、弱类型、基于原型的语言，内置支持类型。它的解释器被称为 JavaScript 引擎，为浏览器的一部分，广泛用于客户端的脚本语言，最早是在 HTML 网页上使用，用来给HTML网页增加动态功能。然而现在 JavaScript 也可被用于网络服务器，如 Node.js。

JavaScript 组成部分

  • ECMAScript
    • ECMAScript是一种脚本语言的标准，ECMA-262标准。
    • 该标准不仅限于JavaScript语言使用，例如ActionScript语言中的标准也为ECMA-262标准
    • ECMAScript 描述了以下内容：语法、类型、语句、关键字、保留字、运算符和对象等。
  • BOM：全称： Browser Object Model，译为浏览器对象模型。
  • DOM：全称： Document Object Model，译为文档对象模型。
